Botanixy

Botanixy is a modern plant e-commerce web application designed to help users discover, purchase, and grow plants with ease. The platform offers a curated collection of indoor plants, seeds, and gardening tools, along with a dedicated plant care guide to support users throughout their plant journey.

The website focuses on a clean, user-friendly interface with interactive features such as categorized browsing, product cards, and visually appealing layouts. It also promotes sustainable living by encouraging users to grow their own plants and adopt eco-friendly practices.

Key features include:
	•	Browse plants by category (indoor, air-purifying, low maintenance, etc.)
	•	Explore a wide range of popular plants
	•	Dedicated “Grow” section for seeds and gardening tools
	•	Plant care guide for beginners
	•	Responsive and modern UI design

Botanixy is built using HTML, CSS, and JavaScript, with a focus on frontend development and user experience.
